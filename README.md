## 说明

>  vue2 + vue-router + vuex + cordova + nodejs 项目实战开发

>  所有的数据通过node request 获取的

>  前后端分离的架构设计，配置webpack的反向代理实现前后端数据的通信.

>  结合cordova api实现硬件访问功能，通过phonegap开发者预览工具进行测试

>  这个项目有两个输出产物，一个是单页面的webapp,一个是hybridapp
